## ✨ Features

- 🎮 **Custom Mobs** — Define any mob with custom stats, AI, equipment and drops
- 🤖 **3 AI Modes** — BASIC, ADVANCED (phases), EPIC (full boss mechanics)
- 🌀 **Custom Dimensions** — Create dimension worlds with exclusive mobs
- 🔮 **Portal System** — Custom portal frames, activation keys, auto-generation
- 💀 **Boss System** — Boss bars, phase transitions, summoner items, countdown timers
- 🖥️ **GUI Management** — Full in-game GUI to create and edit mobs without editing files
- 🌍 **Multilanguage** — Spanish and English built-in

## 📦 Quick Install

1. Download the latest `.jar` from [Releases](../../releases/latest)
2. Drop it in your server's `plugins/` folder
3. Start the server — config files generate automatically
4. Use `/cm gui` to manage mobs in-game

## 📋 Requirements

| Requirement | Version |
|---|---|
